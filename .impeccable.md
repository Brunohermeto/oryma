# Oryma — Design Context

## Design Context

### Users
Equipe de gestão da MCL Informática LTDA (marca RAGALUMA), empresa de e-commerce B2C de produtos de bebê, vendendo em Mercado Livre, Shopee e Amazon. Usam a plataforma no escritório durante o dia para fechar DRE, analisar margens, acompanhar importações. Pequeno time (2-5 pessoas). Usuários operacionais e donos do negócio.

### Brand Personality
A plataforma chama **Oryma** — ferramenta proprietária de inteligência financeira para marketplaces.
- **Vivo**: vibrante, alegre, não sombrio
- **Preciso**: orientado a dados, eficiente
- **Capaz**: sensação de ferramenta séria e profissional

Objetivo emocional: o usuário abre o sistema e sente "tenho controle total + isso é impressionante".

### Manual de Marca Oficial
Tagline: "Inteligência financeira para marketplaces"

**Paleta de Cores:**
- `#0B1023` — Azul Noite (sidebar, textos escuros)
- `#125BFF` — Azul Elétrico (cor primária de marca)
- `#00D6FF` — Ciano (acento secundário)
- `#7B61FF` — Violeta (acento terciário)
- `#E6EAF2` — Prata (backgrounds claros)

Gradiente de marca: `linear-gradient(135deg, #125BFF 0%, #00D6FF 50%, #7B61FF 100%)`

**Tipografia:**
- Títulos / headings: **Sora SemiBold** (font-family: var(--font-sora))
- Corpo / corpo de texto: **Inter Regular** (font-family: var(--font-inter))
- Números precisos / dados: **Geist Mono** (font-family: var(--font-geist-mono), class: .num)

**Ícone da marca:** SVG circular com nós de dados e linha horizontal — gradiente #125BFF → #00D6FF → #7B61FF

### Aesthetic Direction
**Light mode** — confirmado. Sidebar dark (#0B1023), conteúdo em fundo claro (#E6EAF2-like).

**Tema: "Computational Warmth"** — tecnológico preciso mas com calor nos acentos.

**OKLCH colors in code (hue 258°):**
- Background: `oklch(0.96 0.010 258)`
- Border: `oklch(0.88 0.016 258)`
- Muted text: `oklch(0.50 0.025 258)`
- Card bg: white
- Primary brand: `#125BFF` (Azul Elétrico)
- Sidebar bg: `#0B1023` (Azul Noite)

### Design Principles
1. **Dados conectados** — qualquer número pode ser clicado para ver sua origem
2. **Densidade legível** — muito dado na tela, mas hierarquia clara
3. **Brand fiel** — sempre usar a paleta oficial do manual de marca
4. **Ferramenta, não decoração** — cada elemento earn seu espaço
5. **IA integrada** — Oryma Intelligence flutuante em cada tela

### AI Integration
- **OrymaIntelligence** — componente flutuante no canto inferior direito de todas as páginas do dashboard
- Botão: gradiente de marca (#125BFF → #00D6FF → #7B61FF) com ícone Sparkles
- Chat modal: cabeçalho #0B1023, mensagens do usuário em #125BFF, respostas em branco
- Sugestões contextuais por tela
- API: `/api/ai/chat` — requer ANTHROPIC_API_KEY no .env.local

### Sidebar Structure
Sidebar (#0B1023) dividida em 3 seções:
- **Principal**: Dashboard, DRE Gerencial, Painel Tributário
- **Produtos**: Custo por Produto, Velocidade de Venda, Simulador de Preço
- **Operacional**: Feed de Vendas, NF-e / Importações, Despesas Operacionais, Configurações

Item ativo: fundo #125BFF com texto branco + indicador vertical esquerdo
Item hover: rgba(255,255,255,0.07) fundo + rgba(255,255,255,0.90) texto

### Anti-patterns (NUNCA usar)
- border-left como stripe colorida em cards (absolute ban)
- Gradient text (background-clip: text)
- Glassmorphism decorativo
- Emojis como ícones de navegação
- Cores sem relação com o manual de marca
- Cyan-on-dark genérico de startup
